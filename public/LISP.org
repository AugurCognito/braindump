*** =quote= or ='(list)=
=quote= returns a list with evaluating it, important if we want to give a list as an argument to a function.
#+begin_src lisp :results value verbatim
(quote (* 2 6))
#+end_src

#+RESULTS:
: (* 2 6)
='= is just a way to write =(quote (list))=
#+begin_src lisp :results value verbatim
'(* 2 6)
#+end_src

#+RESULTS:
: (* 2 6)
